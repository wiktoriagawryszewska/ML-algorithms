# Machine Learning Algorithms

This repository contains implementations of several machine learning and algorithmic exercises created during my Computer Science studies.

## Algorithms included

- **KNN (K-Nearest Neighbors)** – basic implementation of the KNN classification algorithm  
- **Sequential Covering** – rule-based learning algorithm  
- **LEM2** – algorithm used in rough set theory  
- **Pygame project** – small experiment using the Pygame library  
- **Robotics script** – algorithmic exercises related to robotics

## Technologies

- Python
- Basic Machine Learning concepts
- Algorithm design

## Purpose

The goal of this repository is to practice implementing machine learning algorithms from scratch and better understand how they work internally.
